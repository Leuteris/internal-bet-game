Game format
  • For the old dogs, it's the same format we had for Euro 2016. For the newcomers, please read scheme below
    • User selects 1/X/2 in all matches (not the exact score)
    • If you miss the pick, you get 0 points.
    • If you make the pick, you get the "base points" multiplied by the "betting factor".
      • What is "base points"? Base points = 100 for all Group Stage matches, 200 for all the rest of the matches except for the Final that gives 300.
      • What is the "betting factor"? It's the factor rewarding risky bets vs safe ones? Example:
        • Let's say England vs Italy gives 1.5 for "1", 2.5 for "X" and 3.5 for "2".
          • If you've selected "1" and you guessed it right you'll get Base Points * 1.5
          • If you've selected "X" and you guessed it right you'll get Base Points * 2.5
          • If you've selected "2" and you guessed it right you'll get Base Points * 3.5
          • Again: if you've guessed it wrong you get 0.
      • Betting factors will be based on real ones from a betting house that will be communicated. Yes, we are taking ourselves seriously.
  • Users will also select Over/Under in all matches except for the Group Stage.
    • This gives extra points ONLY if the 1/X/2 pick was correct
    • Extra points given by guessing right is: 0.5 * Base Points * "O/U betting factor"
      • Yes, from a betting house
  • Note: All results are "90-min period" results (any overtime will be recorded as a tie = "X"); applies both on 1/x/2 and O/U
  • Most points win!

Phases
Users will make picks in five (5) phases (therefore bets will be in chunks)
  • Group Stage.
  • Round of 16
  • Quarter Finals
  • Semi Finals
  • Finals for 1st and 3rd position
All betting factors will be retrieved from bet365.com, updated on the latest data before each new phase begins, but will be locked in due time.

Terms
  • This is not a corporate event and it is not whatsoever related or affiliated with the company; it's an event among friends that happen to be colleagues
  • In case of missing picks after the deadline, a random number generator will submit them on your behalf. There will be no exceptions whatsoever
  • We will not build T&Cs to foresee handling of unexpected events; if something comes up, we will just take a subjective decision based on what we consider fair; or we will just take the money and run
  • If you don't like the setup, do not join

Privacy policy
For user experience purposes, the following information per participant will be stored in the system: name, surname and email. These data will be inserted manually by the system administrator upon every participation confirmation (payment confirmation) and will be used for authentication to the service, account administration and user identification in the competition's leaderboard, chat and analytics features. No data will be shared with any 3rd party.
Each Participant has the right to ask for account deactivation and deletion of his/her data. The request will be processed within one month after the end of the competition.
By participating to World Cup Challenge 2018 you agree with the privacy policy.
